# Publicación recomendada

## Mi recomendación

- Frontend y hosting: Cloudflare Pages
- Imágenes futuras: Cloudinary
- Ventas futuras: Shopify Buy Button o Shopify

## Configuración para Cloudflare Pages

- Framework preset: `Vite`
- Build command: `npm run build`
- Build output directory: `dist`
- Root directory: vacío
- Production branch: `main`

## Dominio `.com`

1. Agregar el repo a Cloudflare Pages
2. Deploy inicial con `pages.dev`
3. Ir a `Custom domains`
4. Agregar `tudominio.com`
5. Agregar `www.tudominio.com`
6. Elegir uno como principal
7. HTTPS queda automático
